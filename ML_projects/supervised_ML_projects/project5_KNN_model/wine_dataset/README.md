# Project 5: KNN Model

This project demonstrates a K-Nearest Neighbors (KNN) classifier.  
- **Goal:** Predict outcomes based on nearest neighbors.  
- **Libraries:** scikit-learn, pandas, numpy.  
- **Steps:** Data preprocessing → Model training → Evaluation.  

## How to run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the notebook: `jupyter notebook Project5_KNN_Model.ipynb`
